# Finder API

This project was created using `bun init` in bun v1.0.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## How to Run
* Clone repository

* Generate .env
```bash
cp .env.example .env
```

* Setup your database in env file

* Install dependencies:

```bash
bun install
```

* Run:

```bash
bun run src/app.ts
```
